Подсказка 1
Свойство list-style-type влияет на вид маркера списка. Оно изменяет тип маркера или удаляет маркер для каждого элемента списка. Это свойство может принимать следующие значения:

disc
диск
circle
круг
square
квадрат
decimal
целые числа
lower-roman
строчные римские цифры (i, ii, iii, iv)
upper-roman
заглавные римские цифры (I, II, III, IV)
upper-alpha
большие латинские буквы (A, B, C, D)
lower-alpha
маленькие латинские буквы (a, b, c, d)
none
маркер отсутствует
